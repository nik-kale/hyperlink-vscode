## [1.0.0] - 2024-11-23

### Added
- Initial release of Hyperlink Theme for VS Code
- Complete port of Stef Kors' Hyperlink theme from Nova editor
- Dark blue background (#1C2639) with lighter UI elements (#28344B)
- Comprehensive syntax highlighting with vibrant color palette:
  - Yellow/Gold (#FDD46B) for keywords, strings, and boolean values
  - Cyan (#53E0FC) for types, classes, and HTML tags
  - Orange/Red (#F08261) for variables and arguments
  - Green (#B8E97B) for functions and methods
  - Purple (#C67BE9) for properties
  - Light Gray (#A8ADB7) for comments
  - White (#FFFFFF) for text and operators
- Customized terminal colors matching the editor theme
- Full UI theming for consistent experience across VS Code
- Support for multiple programming languages
- Optimized contrast ratios for reduced eye strain
- MIT License
- Comprehensive README with installation instructions
- Professional documentation (CONTRIBUTING.md, CODE_OF_CONDUCT.md)
- GitHub issue templates and PR template

### Features
- Zero-bloat design focusing on essential color definitions
- Semantic color assignments for intuitive code reading
- Carefully balanced colors for long coding sessions
- Consistent styling across different file types
